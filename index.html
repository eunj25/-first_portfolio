Publishing skill

Html

웹 표준과 웹 접근성을 준수하는 html 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다. 웹표준을 위해 html5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃 설계합니다. 퍼블리싱 하기 전에 ‘기능정의서’를 만들고 웹사이트의 전체적인 폰트, 컬러 가이드, 사이즈 정보, 상호작용 여부, 와이어프레임까지 정리해 놓고 클래스 네임과 함께 HTML 정보 구조설계 작업까지 정리해 꼼꼼하고 체계적인 퍼블리싱 작업을 수행합니다.

css

CSS 코딩은 IE 지원도 충분히 고려하는 크로스브라우징(cross browsing)을 위해 안정적인 CSS 속성을 사용해서 HTML Elements를 배치하고 디자인합니다. 특별한 경우 CSS3에서 새롭게 지원하는 속성을 활용해서 조금 더 다양한 CSS배치과 디자인을 합니다.

js

웹 퍼블리싱에 필요한 네비게이션, 탭 메뉴, 어코디언 등 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수 제이쿼리 코딩으로 제작합니다.필수적인 자바그크립트 활용은 가능하며 주로 제이쿼리와 병행해서 활용합니다.

Responsive web

반응형 웹 제작에 있어서 크로스브라우징을 위해 안정적인 기존 방식인 float, position, media query를 활용한 반응형 웹 제작 경험을 충분히 가지고 있습니다. 필요에 따라 flex와 grid 방식으로 반응형 웹 사이트 제작도 충분히 활용해서 제작합니다.

Mobile web app

모바일 우선을 위해 mobile first에 준하는 모바일 웹 앱 제작 경험도 충분히 가지고 있습니다. UI/UX 측면에서 미적인 디자인 뿐만 아니라 모바일 사용자 경험에 최적화를 위해 효율적인 컨텐츠 배치를 우선적으로 생각합니다. 모바일 대응을 위한 폴랫폼으로 반응형 웹, 모바일 웹에 대한 충분한 이해도를 갖고 있습니다.

Photoshop, Illustrator, Premiere

웹사이트 UI 디자인 작업은 포토샵을 활용하며 웹 UI 디자인 작업을 위한 포토샵 사용 능력은 충분하다고 생각합니다. 이벤트 디자인, 배너 디자인, 썸네일 제작 등 디자인이 가능하며, 인물보정을 위한 포토샵 사용도 가능합니다. 로고 제작은 일러스트 사용에 익숙합니다. 영상편집은 컷 편집, 자막, 이미지 삽입, 음향편집 등 기본적인 작업을 할 수 있습니다.

Text editor

웹 퍼블리셔로서 텍스트에디터는 visual studio code를 메인으로 사용하며 경우에 따라 brackets를 병행해서 사용합니다. 간단한 부분 수정을 위해 editplus, notepad++도 활용합니다.

'습득력이 빠르며 기본기가 잘 돼있는 열정적인 웹 퍼블리셔 신은주입니다.’

폴더 구조, 경로를 체계적으로 정리하며, 파일명과 폴더명 구분이 쉽게 하여 퍼블리싱 하기 수월하게 해 놓습니다. 퍼블리싱 작업할 때에도 주석 처리 하면서 인텐트와 함께 가독성을 높여 작업하는 것에 습관화되 있습니다. 또한, 퍼블리싱 작업 전 와이어프레임 및 기능정의서, HTML 구조설계를 정리하여 체계적인 구조설계를 진행합니다. 체계적인 과정으로 원활한 작업이 가능합니다.

기능성을 제외한 모든 작업은 하드코딩을 원칙으로 탭 메뉴 콘텐츠 등 제작 시 순수 CSS로 작업합니다. 그리드 레이아웃 작업에서 Frameworks 그리드를 사용하지 않고 Position과 float 기존방식, CSS Flex와 CSS Grid를 충실히 사용합니다.
